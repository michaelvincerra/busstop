TriMet Bus Stop Locator



##Table of Contents

- TrovaTrasporto: Public Transportation Finder using TriMet API in Portland, Oregon. 
- Installation: On Hold.
- Requires use of Google Maps API and TriMet API. Geolocation coordinates are derived from user's geographical position: Portland, Oregon, metro area only. 
- Clearing Logic: Resets available stops and arrival times based uponn a user's physical location. 



Background

TrovaTrasporto: Provides user with Estimated Arrival Times, busstop/tram locations, and links to bus/tram schedules for public transportation in Portland, Oregon, based on user's present location. Project initially developed as part of a code challenge assignment at the PDX Code Guild, Portland, Oregon.
